居中定位样式

一、文字垂直居中

CSS

                        
.parent{
    line-height:200px;
}
.parent .children{
    vertical-align:middle;
}

1.通过table和table-cell来实现,子类元素会撑得跟父类元素一样高
内敛元素垂直居中
innerText垂直居中

2.通过line-height=height来实现,内敛元素需要设置为inline-blocks

innerText垂直居中
内联元素垂直居中

二、块状元素垂直居中

1.子类absolute定位，top50%,translate3d(0,-50%,0)，父类非static定位

内敛元素垂直居中
innerText不垂直居中

2.父类设置为弹性盒布局，并设置交叉轴垂直居中,所有子类， 包括inner-text均被设置为block，text-align:center不奏效

内敛元素垂直居中
innerText垂直居中

三、块状水平居中和垂直居中， 内联元素直接设置text-align:center就可以实现水平居中

1.子类absolute定位，top50%,left:50,translate3d(-50%,-50%,0)，父类非static定位

内敛元素垂直居中
innerText不垂直居中

2.父类容器设置margin:auto和position:absolute

内敛元素不垂直居中
innerText不垂直居中

3.父类设置为弹性盒布局，并设置主轴和交叉轴垂直居中

内敛元素垂直居中
innerText垂直居中
